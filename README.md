# twitter_clone

A simple react UI clone of twitter with the following feature :
1. Post a tweet.
2. Delete a tweet.
3. Add comment.
4. Delete comment.
5. Like posts.

Link to hosted site : https://twitter-clone-fb54a.web.app/
